# react grab anywhere injector

[![Install with TamperMonkey](https://img.shields.io/badge/Install%20with-TamperMonkey-blue.svg?logo=Tampermonkey&logoColor=white)](https://github.com/Microck/react-grab-anywhere-injector/raw/main/react-grab-anywhere-injector.user.js)
[![Install with ViolentMonkey](https://img.shields.io/badge/Install%20with-ViolentMonkey-orange.svg?logo=Monkeytype&logoColor=white)](https://github.com/Microck/react-grab-anywhere-injector/raw/main/react-grab-anywhere-injector.user.js)

automatically loads [react grab](https://react-grab.com) into any webpage including `localhost` so you can easily rip elements for **cursor**, **claude code**, **opencode** or anything else.

### installation

1.  install **violentmonkey** or **tampermonkey**
2.  [click to install the script](https://github.com/Microck/react-grab-anywhere-injector/raw/main/react-grab-anywhere-injector.user.js)
3.  refresh your page. if you see `React Grab Anywhere Injected!` you can assume what that means hopefully

### usage

1.  go to any website
2.  hold **cmd+c** or **ctrl+c** and click an element
3.  paste the code into your ai tool

### customization

to restrict this to specific domains just replace the `@match *://*/*` line in the script editor with your preferred patterns. recommended if you don't want this running on your banking site.
